## 这里写什么

主要内容分为前端、爬虫和 AI。随笔不会单独抢占首页，只会偶尔出现在文章时间流里。

文章会持续更新。你可以直接访问网站，也可以通过 RSS 订阅新内容。

文章归档页可以按技术方向筛选。按 `/` 直接开始搜索，按 `Esc` 清空搜索并退出输入。每篇文章都带有目录、阅读进度、分享入口和上一篇、下一篇导航。

## 关于本站

野路子手记没有账户、广告和营销追踪。文章使用 Markdown 保存，网站源码公开在这个仓库中。

如果你发现错字、失效链接或技术问题，欢迎提交 Issue。
